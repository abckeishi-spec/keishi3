AI対話型助成金検索プラグイン - OpenAI GPTを使用した助成金の検索とアドバイス機能

== Description ==

Grant AI Assistantは、AI技術を活用して助成金の検索とアドバイスを提供するWordPressプラグインです。

主な機能:
* OpenAI GPTを使用したAI会話機能  
* 助成金データベースの検索
* リアルタイム対話インターフェース
* ショートコードでの簡単設置
* レスポンシブデザイン対応

== Installation ==

1. プラグインファイルを `/wp-content/plugins/grant-ai-assistant` ディレクトリにアップロード
2. WordPress管理画面の「プラグイン」メニューでプラグインを有効化
3. 「設定」>「Grant AI Assistant」でOpenAI APIキーを設定
4. 投稿や固定ページに `[grant_ai_chat]` ショートコードを追加

== Frequently Asked Questions ==

= OpenAI APIキーはどこで取得できますか? =

OpenAIの公式サイト (https://platform.openai.com/) でアカウントを作成し、API キーを取得してください。

= どのようなショートコードが利用できますか? =

基本的な使用法:
`[grant_ai_chat]`

カスタマイズ例:
`[grant_ai_chat height="600px" title="AI助成金相談"]`

= Grant Insight Perfectテーマ以外でも使用できますか? =

はい、どのWordPressテーマでも使用できます。ただし、助成金投稿タイプが存在する場合により効果的に動作します。
